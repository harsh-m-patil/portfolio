EZstay is a full-stack hotel booking platform built to simplify the reservation process and deliver a smooth user experience.

Using EJS, Express, MongoDB, and vanilla JavaScript, EZstay delivers a fast, accessible UI and clean backend functionality to manage listings, bookings, and user interactions.

## Features

- Search functionality with location-based filters
- Real-time availability and booking management
- EJS-rendered templates for a dynamic, server-rendered UI
- Intuitive design to enhance usability and responsiveness

## Results

- Query time reduced by **25%** with optimized search filters
- User satisfaction improved by **15%** after UI redesign
