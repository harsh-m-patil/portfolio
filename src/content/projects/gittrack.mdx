GitTrack is a command-line interface tool built in Go using Cobra and the GitHub API. It allows developers to instantly track any user's GitHub activity, repositories, and contributions.

Perfect for productivity-focused devs, GitTrack enhances your workflow by bringing GitHub stats directly to your terminal.

## Features

- Track user activity, contributions, and starred repos
- Fetch and display live repo stats via GitHub API
- Built with Go and Cobra for blazing fast execution
- Minimal, user-friendly CLI with clear output

## Impact

- Supported by the GitHub API, scalable to **100M+ users**
- Helps developers quickly monitor their own or others' open-source activity
